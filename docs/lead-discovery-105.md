## Product Positioning

GENESIS is an AI Code Safety Gate.

It sits between AI-generated code and execution.

It classifies changes as:

- ALLOW
- REQUIRE_APPROVAL
- BLOCK

One-line message:

Stop risky AI code before it runs.

A strong lead has at least one of these public signals:

- Posted about Cursor
- Posted about Copilot

## Lead Scoring

Score every lead from 1 to 5.

### Score 5 — Very Strong

The person publicly described a real AI coding failure, workflow concern, or need for safer AI-generated code.

Action:

Prioritize for first 10.

### Score 4 — Strong

The person uses AI coding tools publicly and works in devtools, engineering, CI, or code review.

Action:

Likely first 10 candidate.

### Score 3 — Medium

The person discusses AI coding tools but no clear pain signal yet.

Action:

Add to backup list.

### Score 2 — Weak

The person is technical but the AI coding relevance is unclear.

Action:

Do not contact in first batch.

### Score 1 — Poor Fit

No relevant developer or AI coding signal.

Action:

Do not contact.

## Outreach Readiness Rule

A lead is CONTACT_READY only if:

- public relevance is clear
- the outreach angle is specific
- the message can be personalized
- the user can manually verify the person
- no sensitive private data is used
- no spam pattern is created

## First Batch Rule

Do not contact 30 people.

Discover 30.

Shortlist 10.

Send to 3 first.

Review response quality.

Then decide whether to continue to 10.

## Safety Rules

Do not:

- automate sending
- mass-message
- scrape private data
- collect sensitive personal data
- use fake identity
- imply a personal relationship
- overstate product maturity
- claim enterprise readiness
- mention private GENESIS core
- mention payment before pain
- send checkout links
- send generic spam

## Manual Outreach Rule

The user must manually approve each first batch message.

## Success Criteria

Minimum:

- 30 public leads discovered
- 10 leads shortlisted
- 3 first messages prepared
- 0 automated sends
- 0 spam behavior

Strong:

- 30 public leads discovered
- 10 strong shortlisted leads
- 3 highly personalized messages prepared
- 1 real pain signal after manual sending

## Decision Rule

If no pain signal appears after first outreach:

Improve targeting before building more product.
