# GENESIS — Manual Launch Checklist

## Live Domain

- Live demo: https://aicodesafety.com
- Public repo: https://github.com/aicodesafety/genesis-ai-code-safety-demo

## Before Posting

- Public repo exists
- README is polished
- Demo commands work
- Live domain confirmed working
- GitHub topics added
- Repo pinned on profile
- Outreach tracker ready
- Pilot qualification checklist ready

## GitHub

- Add description
- Add topics
- Enable Issues
- Pin repo on profile
- Add live demo link to About section: https://aicodesafety.com
- Confirm private core is not visible publicly

## LinkedIn

- Copy launch post from docs/social-linkedin.md
- Post with repo link
- Ask one clear feedback question
- Reply to every comment
- Record serious replies in feedback tracker

## X

- Copy one variant from docs/social-x.md
- Post repo link
- Ask what risky AI code patterns people have seen
- Reply quickly

## Direct Outreach

- Choose 10 people
- Send outreach message
- Track replies
- Ask discovery questions
- Do not hard sell too early

## Reddit / Hacker News

- Wait until GitHub + LinkedIn + direct outreach are done
- Choose one community only
- Read rules first
- Lead with problem/question
- Add repo link after context

## Pilot

- Do not offer pilot to everyone
- Qualify pain first
- Offer pilot only if they use AI coding tools and have real risk
- Track candidates

## Daily Review

At end of each day, answer:

1. How many people saw it?
2. How many replied?
3. What pain repeated?
4. Did anyone ask for a demo?
5. Is anyone a pilot candidate?
6. What should change in positioning?
